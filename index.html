Projects

Bed and Breakfast Website

I created a fully responsive website for a Bed and Breakfast using HTML and Bootstrap. This project involved designing an intuitive user interface that highlights the charm and amenities of the establishment. The site features a photo gallery, room descriptions, booking form, and local attractions, providing potential guests with all the information they need.

Class Registration Form

I developed a registration form for selecting classes at Ozarks Technical Community College (OTC). This project focused on creating a user-friendly form with HTML, ensuring that students can easily register for their desired classes. The form includes fields for personal information, course selection, and submission validation.

Course Schedule Display Page

For OTC, I also created a display page for my course schedule. This project involved organizing course information into a clean, easy-to-read format. The page allows students to view their schedule at a glance, including course names, times, and locations, ensuring they have all the necessary details for their academic planning.
